# 🌐 Interactive Social Media Icons (Menu de Redes Sociais)

> Um componente de interface elegante e responsivo, desenvolvido com HTML5 e CSS3, apresentando um conjunto de ícones de redes sociais com foco em interatividade e micro-interações.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma barra de ícones de redes sociais que sirva como um ponto de contato intuitivo para o usuário. 

O foco técnico aqui é a utilização de fontes icônicas para garantir que o design permaneça leve, escalável (vetorial) e de fácil integração em qualquer layout web profissional.

## ✨ Destaques e Aprendizados Técnicos

* **Integração de Bibliotecas Modernas:** Uso da CDN do **Font Awesome (v5.15.2)** para implementar ícones de marcas globais (Facebook, Twitter, Instagram e YouTube) de forma performática.
* **Arquitetura de UI Modular:** Estruturação dos elementos através de uma classe contêiner (`.social_media`) dentro de uma `div` principal, facilitando o posicionamento e a aplicação de estilos globais aos links.
* **Foco em Micro-interações:** Estrutura preparada para efeitos de *hover* personalizados, demonstrando atenção aos detalhes que enriquecem a Experiência do Usuário (UX).
* **Acessibilidade e Semântica:** Uso correto de tags de âncora (`<a>`) para garantir que os ícones funcionem como links de navegação reais.
* **Organização Profissional de Código:** Separação estrita entre a estrutura semântica (`index.html`) e a identidade visual localizada no diretório `css/style.css`, seguindo os padrões recomendados de engenharia de software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação e organização de elementos de link.
* **CSS3:** Estilização, gerenciamento de cores e design responsivo.
* **Font Awesome (v5.15.2):** Iconografia vetorial para suporte visual e branding.
